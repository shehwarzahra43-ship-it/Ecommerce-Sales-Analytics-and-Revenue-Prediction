# 📊 E-Commerce Sales Analytics and Revenue Prediction

## 📌 Project Overview

This project focuses on the analysis of an e-commerce sales dataset and the development of a machine learning model to predict sales revenue. The dataset contains customer transactions, product information, order details, payment methods, and sales values collected from an online retail environment.

The project combines Exploratory Data Analysis (EDA), data visualization, and Linear Regression to uncover business insights and predict the total value of customer purchases.

---

## 🎯 Objectives

The primary objectives of this project are:

- Analyze customer purchasing behavior.
- Identify monthly sales trends.
- Compare revenue generated by different products.
- Examine order status distribution.
- Explore relationships between transaction variables.
- Develop a predictive model for estimating TotalPrice.
- Evaluate the performance of the regression model.

---

## 📂 Dataset Information

The dataset contains **1,200 e-commerce transactions** and **14 variables**.

### Dataset Features

| Feature | Description |
|----------|-------------|
| OrderID | Unique identifier for each order |
| Date | Date of purchase |
| CustomerID | Unique customer identifier |
| Product | Product purchased |
| Quantity | Number of units purchased |
| UnitPrice | Price per unit |
| ShippingAddress | Customer shipping address |
| PaymentMethod | Payment method used |
| OrderStatus | Status of order (Delivered, Shipped, Pending, Returned, Cancelled) |
| TrackingNumber | Shipment tracking number |
| ItemsInCart | Number of items added to cart |
| CouponCode | Promotional coupon code |
| ReferralSource | Marketing source through which customer arrived |
| TotalPrice | Total value of the transaction |

---

## 🧹 Data Preprocessing

The following preprocessing steps were performed:

- Loaded the dataset using Pandas.

- Handled missing values in the `CouponCode` column.
- Replaced missing coupon values with `"NoCoupon"`.

## 📈 Model Performance

### R² Score = 0.88

The regression model achieved an **R² score of 0.88**.

### Interpretation

The model explains approximately **88% of the variance in TotalPrice**, indicating a strong predictive relationship between the selected input features and sales revenue.

This result demonstrates that Quantity, UnitPrice, and ItemsInCart are strong predictors of transaction value and that the model provides high predictive accuracy.
